# Install Prerequisites for Python3 venv support


=== "MacOS"
    ```shell
    echo "python3 venv ready"
    ```

=== "Linux"
    ```shell
    ---
    validate: python3 -m venv -h
    ---
    apt-get -y install python3-venv
    echo "python3 venv ready"
    ```

=== "Win32"
    ```shell
    echo "python3 venv ready"
    ```
